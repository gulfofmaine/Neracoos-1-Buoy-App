# Current Conditions

[`src/Features/ERDDAP/Platform/Observations/CurrentConditions`](https://github.com/gulfofmaine/Neracoos-1-Buoy-App/blob/master/src/Features/ERDDAP/Platform/Observations/CurrentConditions/index.tsx) - [Platform M01 - Current Conditions](https://mariners.neracoos.org/platform/M01)

The current conditions page is designed to give the user a quick view
of the last 24 hours of data.

It honors the set unit system, and if you switch
unit system while it is displayed it will smoothly transition.
To see the transition, switch to the **Canvas** tab and select the **Configurable** story.

_Note: the Wind does not display correctly in Storybooks_
